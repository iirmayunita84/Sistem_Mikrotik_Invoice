Sistem Mikrotik Invoice

Aplikasi untuk membuat invoice dan struk pembayaran pelanggan MikroTik

Deskripsi

Sistem Mikrotik Invoice adalah aplikasi desktop berbasis Python (Tkinter & ttkbootstrap) yang memudahkan pembuatan invoice, struk pembayaran, dan manajemen pelanggan MikroTik. Fitur utama termasuk:

  • Membuat invoice PDF otomatis
  • Preview & cetak struk thermal 80mm
  • Edit comment pelanggan langsung ke MikroTik
  • Manajemen data pelanggan manual
  • Integrasi dengan MikroTik API (opsional)

Cara Instalasi

1. Pastikan Python 3.x sudah terpasang.
2. Install dependencies: 

pip install reportlab pillow pdf2image pywin32 ttkbootstrap routeros-api

3. Jalankan aplikasi:
python main.py
